# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities **privately**, not through public GitHub issues.

- Prefer **[GitHub Security Advisories](https://github.com/g2rain/g2rain/security/advisories/new)** on the main organization repository if you have access.
- Alternatively, contact the maintainers at **g2rain_developer@163.com** with subject line containing `[SECURITY][g2rain-main-shell]`.

Include: affected version, reproduction steps, and impact if known. We will acknowledge receipt and coordinate a fix and disclosure timeline.

## Scope

This policy applies to the **g2rain-main-shell** repository (the Vue 3 + Vite + qiankun main shell / admin framework). Issues in **g2rain-app-template**, **g2rain-app-cli**, or **generated sub-apps** may be reported the same way; please state which component is affected.
